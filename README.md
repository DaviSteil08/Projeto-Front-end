# 📊 Dashboard: Áreas da Tecnologia da Informação

Este projeto é um site multi-páginas desenvolvido como trabalho acadêmico, com o objetivo de apresentar as principais áreas da TI de forma organizada e acessível. O foco principal é a prática de **HTML5 semântico**, **CSS3** e **versionamento colaborativo com Git**.

## 🎯 Objetivo
Desenvolver um dashboard informativo que sirva como guia para iniciantes conhecerem as diferentes ramificações da carreira tecnológica, aplicando conceitos de design consistente e acessibilidade web.

## 🚀 Tecnologias Utilizadas
*   **HTML5:** Estruturação semântica (header, nav, main, section, footer).
*   **CSS3:** Estilização externa, layout responsivo e menus horizontais.
*   **Git & GitHub:** Controle de versão e colaboração em grupo.

## 📂 Estrutura do Projeto
O site é composto por 8 páginas principais:
*   **Home (`index.html`)**: Introdução e cards de navegação.
*   **Front-end**: Tecnologias de interface e experiência do usuário.
*   **Back-end**: Lógica de servidor e banco de dados.
*   **Dados**: Ciência, análise e engenharia de dados.
*   **Infraestrutura**: Redes e serviços de suporte.
*   **Segurança**: Cibersegurança e proteção de dados.
*   **Mobile**: Desenvolvimento de aplicativos nativos e híbridos.
*   **UX/UI**: Design de experiência e interface.

## ♿ Acessibilidade e Boas Práticas
*   Uso de tags semânticas para melhor leitura por leitores de tela.
*   Atributos `alt` descritivos em todas as imagens.
*   Contraste de cores planejado para legibilidade.
*   Menu estático com indicação de página ativa.
