Projeto Integrador

Uma visão clara, moderna e conectada das principais áreas da TI.

Este dashboard foi reorganizado para apresentar as sete frentes mais relevantes da Tecnologia da Informação com linguagem visual unificada, conteúdo objetivo e navegação intuitiva. O foco é transformar um conjunto disperso de páginas em uma experiência acadêmica com padrão profissional.

7 áreas Conteúdo organizado por especialidade para facilitar estudo, comparação e apresentação.
100% HTML + CSS Estrutura semântica, interface responsiva e componentes reutilizáveis sem JavaScript.
UI consistente Mesma linguagem visual em home, páginas internas, cards, tipografia e sistema de espaçamentos.
Visão Geral

O projeto conecta fundamentos técnicos, mercado e experiência de produto.

Em vez de tratar cada área da TI de forma isolada, o dashboard mostra como elas se complementam na construção de produtos digitais: da interface que o usuário enxerga até a infraestrutura, a segurança e a inteligência baseada em dados que sustentam toda a operação.

Critérios da refatoração

  • Header e footer padronizados em todas as páginas.
  • Design system centralizado com variáveis CSS reutilizáveis.
  • Cards, botões e containers consistentes para toda a navegação.
  • Conteúdo distribuído em blocos claros, legíveis e responsivos.
